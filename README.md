# Todo List App

A simple and interactive Todo List web app built using React and Vite. This app helps users manage daily tasks effectively.

## Features
- Add, delete, and mark tasks as completed.
- Clean and minimalist UI.

## Technologies Used
- React.js
- Vite
- JavaScript
- CSS
